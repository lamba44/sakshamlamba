My Work

Korero

Korero is an innovative project using OpenCV, CVZone, Python, TensorFlow, and Keras to translate American Sign Language (ASL) into text. It empowers individuals with auditory impairments, bridging communication barriers and fostering inclusivity.

Flappy Bird

Introducing my very first game, born from following a Unity tutorial step by step. It's a classic take on the beloved Flappy Bird game, featuring simple controls where you press the spacebar to make the bird jump. Keep an eye on the top left for your score as you navigate through basic challenges. Dive into the fun and nostalgia of this classic gaming experience!

Dictionary Game

A fun and interactive web-based game that challenges users to enter as many valid words as possible within a selected time limit. The game interface is designed with a clean and responsive layout, featuring a sleek design using the Poppins font. Players can choose from different timer durations, enter words, and see their score updated in real-time. The project integrates Google Analytics for tracking user engagement and Firebase for streamlined app configuration and analytics.
